Menu Ressource

    New resource
    Permet de créer une nouvelle ressource dans la table de ressources courante. Une fenêtre de dialogue permet de spécifier le type, le numéro ID, le nom et les attributs de la nouvelle ressource. Une ressource est alors créée ne contenant aucune donnée. Noter que cette ressource est créée en mémoire et qu'il faudra sauvegarder le fichier pour la rendre définitive.

    Edit
    Appelle un éditeur pour la ou les ressources sélectionnées dans la table de ressources courante. Si Rezilla ne trouve aucun éditeur dédié, il cherche si un gabarit est disponible afin d'invoquer l'éditeur de gabarits. En cas d'échec, il appelle l'éditeur hexadécimal. C'est équivalent à double-cliquer sur les ressources dans la table.

    Template Editor
    Appelle l'éditeur de gabarits pour la ou les ressources sélectionnées dans la table de ressources courante. Si Rezilla ne trouve aucun gabarit, il appelle l'éditeur hexadécimal. C'est équivalent à double-cliquer sur les ressources dans la table avec la touche Majuscule enfoncée.

    Hexadecimal Editor
    Appelle l'éditeur hexadécimal pour la ou les ressources sélectionnées dans la table de ressources courante. C'est équivalent à double-cliquer sur les ressources dans la table avec la touche Option enfoncée.

    Edit as Type...
    Force Rezilla à utiliser un type de remplaceement. Dans le dialogue affiché, on peut spécifier le type soit à partir d'un menu, soit en le tapant directement dans le champ de saisie.

    Remove
    Supprime toutes les ressources sélectionnées de la table de ressources.
    touche Option enfoncée.

    Duplicate
    Duplique la ou les ressources sélectionnées dans la table de ressources. Cette commande attribue un numéro unique d'identification (ID) aux ressources dupliquées. On peut modifer ces valeurs par la suite dans la fenêtre Inspecteur.
