Panneau Export

Dans le panneau Export, on peut spécifier l'encodage à utiliser pour les données binaires: encodage Base64 ou représentation hexadécimale. La case à cocher Include binary data in output permet de spécifier si les données binaires de chaque ressource doivent être inclues dans les fichiers d'export: cela concerne uniquement les formats Texte et XML. Pour plus d'informations, voir la section Export et import des tables de ressources.

Dans le champ appelé Text editor signature, on peut indiquer le créateur d'un éditeur de textes à attribuer aux fichiers produits afin que, par la suite, ceux-ci soient automatiquement ouverts dans cet éditeur lorsqu'on fait un double-clic sur leur icone.
